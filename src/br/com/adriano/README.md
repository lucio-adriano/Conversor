# Conversor
===============


1. Tela do Sistema (Menu Conversor).

   Escolha umas das opções disponivel para realizar a conversão em seguinda click em "OK".
   
   ![Menu Conversor](C:/Aulas/alura/Conversor/Conversor-moeda/src/br/com/adriano/assets/telas/menu-conversor.png "Meu Conversor")
   
   Digite um valor em Real R$, não colocar "," casa decimal representado por "."(padrão americano) em seguinda click em "OK".   
   
   ![Entrda de valor](C:/Aulas/alura/Conversor/Conversor-moeda/src/br/com/adriano/assets/telas/informe-valor.png "Entrda de valor")
   
   
   Escolha uma das opções de moedas para realizar a conversão em seguinda click em "OK".
   
   ![Escolha Moeda](C:/Aulas/alura/Conversor/Conversor-moeda/src/br/com/adriano/assets/telas/escolha-moeda.png "Escolha Moeda")

   Escolha "Yes" para continuar "No" ou "Cancel" para sair.
   
   ![Escolha Moeda](C:/Aulas/alura/Conversor/Conversor-moeda/src/br/com/adriano/assets/telas/continuar-sair.png "Escolha Moeda")
